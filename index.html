About

Hi, my name is Dennis and I am a Computer Science student at York University.

I'm not particularly an expert in any given field, though through my degree I've become instilled with different methods to solving problems through greedy and dynamic approaches. For example, using a greedy approach for a knapsack problem (where fractions are allowed), or using dynamic programming for the standard max-flow problem.

Languages: I am most comfortable with Java and JavaScript; however, I am well acquainted with C, C++, and basic shell commands. I prefer an Object-Oriented approach to programing which lead me to pick up C# for my game design projects. Lastly, I am very confident in my HTML and CSS which I love to utilize towards making responsive web-apps.

I've developed lots of projects from my schooling, from simple android apps to online record shops that use SQLite to access and maintain a database. I developed a Regular Expression Evaluator through Eiffel which takes a contract (client-supplier relationship) approach to programming. The program required allowed my group and I to apply many programming principles like single choice, uniform access, and a composite-visitor pattern. For working with databases, I have strong knowledge in relational algebra and am continuing to learn SQL and PostgreSQL. In the end, I hope to keep expanding my knowledge and keep up with the everchanging technology.

Word Replacement

The site takes the input of the user and makes an HTTP Get request to Datamuse's API. I divide the input into an String Array seperated by non-characters using regex. Using the array I iterate through the each word and make the respective query to the API and display the result in the Response Field. I aim to use promises to preserve the order as some words return faster then others.
